# Dark Bruv Neo

> A premium VS Code theme engineered for AI-first development workflows.

Dark navy background. Signature pink accents. Full semantic token support across 12+ languages — built for developers who spend all day in the editor.

---

## What's New in v2

- `semanticHighlighting` enabled — colors powered by language servers, not just regex
- Full token hierarchy: keywords → types → functions → variables → strings → comments
- Expanded language coverage: TypeScript, TSX/JSX, Python, Go, CSS/SCSS, JSON, Markdown, Shell, SQL
- Decorators, generics, interfaces, and imported symbols all have dedicated colors
- Contrast-optimized palette for long coding sessions

## Install

Search `Dark Bruv Neo` in the VS Code Extensions panel, or:

```
ext install themebruv.dark-bruv-neo
```

Then open Command Palette → `Preferences: Color Theme` → select **Dark Bruv Neo**.
